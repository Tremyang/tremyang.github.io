Windows端JDK安装及配置

发表于 2019-06-07 更新于 2020-03-21 分类于 Java

1.下载JDK

前往oracle官网下载页面
下载页面

3.配置环境变量

通过控制面板–>系统和安装–>系统 进入页面

点击环境变量

在系统变量内新建JAVA_HOME,变量值为JDK的安装目录

编辑系统变量内的Path变量

添加变量值
